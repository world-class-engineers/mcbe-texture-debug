# JSON UI Data Encoding

Generated by JSON UI Interface Generator

## Encoding Scheme

Data is encoded as a single string with:

1. Prefix for title_text (`__collection_form__`)
2. All fields (number and string) encoded into a header, alphabetically
3. `§~§r` separator (4 bytes)
4. Display text (what the button actually shows)

**Note:** The `_prefix` field is used when encoding data for `#title_text` to help locate your data in the title string.

### Byte Layout

| Field       | Type   | Prefix                                | Digits/Length | Byte Range |
| ----------- | ------ | ------------------------------------- | ------------- | ---------- |
| activeIndex | number | <code>&#124;activeIndex&#124;:</code> | 2             | 33-35      |
| difficulty  | number | <code>&#124;difficulty&#124;:</code>  | 1             | 48-49      |
| hasNext     | number | <code>&#124;hasNext&#124;:</code>     | 1             | 59-60      |
| hasPrevious | number | <code>&#124;hasPrevious&#124;:</code> | 1             | 74-75      |
| itemCount   | number | <code>&#124;itemCount&#124;:</code>   | 5             | 87-92      |
| pageNumber  | number | <code>&#124;pageNumber&#124;:</code>  | 2             | 105-107    |
| totalPages  | number | <code>&#124;totalPages&#124;:</code>  | 2             | 120-122    |

### Example Encoded String

```
"__collection_form__|activeIndex|:01|difficulty|:1|hasNext|:1|hasPrevious|:1|itemCount|:00001|pageNumber|:01|totalPages|:01§~§rDiamond Sword"
```

- **Header bytes:** 19-122 (activeIndex, difficulty, hasNext, hasPrevious, itemCount, pageNumber, totalPages) (after prefix `__collection_form__`)
- **Separator:** 122-126 (`§~§r`)
- **Display text:** 126+ (visible button text)

## Extraction Bindings

### Field Extraction

All fields (number and string) are encoded into the header before the `§~§r` separator (`$separator`).
To extract a field, use byte-range subtraction:

```
(('%.<END>s' * $var) - ('%.<START>s' * $var))
```

For fields starting at byte 0, the subtraction is unnecessary - just use `('%.<END>s' * $var)`.

### Display String Extraction

The **display string** is the actual button text, located after the header:

```
($var - ('%.<HEADER_LEN>s' * $var))
```

Example: `(#title_text - ('%.122s' * #title_text))`

- **Exists:** `(not ((#title_text - $separator) = #title_text))`
- **Not Exists:** `((#title_text - $separator) = #title_text)`

## Encoder Module

```javascript
/**
 * Encodes data into a JSON UI-compatible string for Minecraft Bedrock
 * @param {Object} data - Field values to encode
 * @param {Object} schema - Schema object with fields at root level
 * @param {string} displayText - The text to display on the button (after the encoded data)
 * @returns {string} Encoded string
 */
function encodeItemData(data, schema, displayText = "") {
  const separator = "§~§r";
  let header = schema._prefix || "";

  const sortedEntries = Object.entries(schema).sort(([a], [b]) => a.localeCompare(b));
  for (const [name, config] of sortedEntries) {
    if (name.startsWith("_")) continue;
    const isNumber = typeof config.default === "number";
    if (isNumber) {
      const value = data[name] ?? config.default ?? 0;
      if (typeof value !== "number" || Number.isNaN(value)) {
        throw new Error("Field " + name + " must be a valid number, got: " + String(value));
      }
      if (value < 0) {
        throw new Error("Field " + name + " cannot be negative, got: " + value);
      }
      if (!Number.isInteger(value)) {
        throw new Error("Field " + name + " must be an integer, got: " + value);
      }
      if (value >= Math.pow(10, config.digits)) {
        throw new Error("Field " + name + " value " + value + " exceeds " + config.digits + " digits");
      }
      const padded = String(value).padStart(config.digits, "0");
      const prefix = config._prefix || "|" + name + "|:";
      header += prefix + padded;
    } else {
      const value = data[name] ?? "";
      const encoder = new TextEncoder();
      const bytes = encoder.encode(value).length;
      if (bytes > config.maxLength) {
        throw new Error("Field " + name + " exceeds maxLength: " + bytes + " > " + config.maxLength + " bytes");
      }
      const prefix = config._prefix || "|" + name + "|:";
      header += prefix + value.padEnd(config.maxLength);
    }
  }

  return header + separator + displayText;
}

module.exports = { encodeItemData };
```

### Usage

```javascript
import { encodeItemData } from "./encoder.js";

// Define your schema
const schema = {
  activeIndex: { default: 0, digits: 2 },
  difficulty: { default: 0, digits: 1 },
  hasNext: { default: 0, digits: 1 },
  hasPrevious: { default: 0, digits: 1 },
  itemCount: { default: 0, digits: 5 },
  pageNumber: { default: 0, digits: 2 },
  totalPages: { default: 0, digits: 2 },
};

// Encode your data with display text
const data = {
  activeIndex: 1,
  difficulty: 1,
  hasNext: 1,
  hasPrevious: 1,
  itemCount: 1,
  pageNumber: 1,
  totalPages: 1,
};

// The encoded string contains your data, then separator, then display text
const buttonText = encodeItemData(data, schema, "Diamond Sword");

// Use in an ActionFormData
const form = new ActionFormData()
  .title("My Form")
  .body("Select an item")
  .button(buttonText, (player, selection) => {
    // The button displays "Diamond Sword" but #title_text contains
    // the full string including your encoded data
  });
form.show(player);
```
